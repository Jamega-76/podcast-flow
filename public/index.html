Configurer Telegram

1
Créer un bot

Ouvrez Telegram, cherchez @BotFather, envoyez /newbot.

2
Copier le Token

BotFather vous donnera un token au format 123456789:ABCdef...

3
Obtenir votre Chat ID

Démarrez votre bot, visitez :
api.telegram.org/bot<TOKEN>/getUpdates
Cherchez "chat":{"id": XXXXX}

4
Tester

Renseignez le token et Chat ID puis cliquez "Sauvegarder".
